# snake
A super minimal TUI snake game written in pure BASH v5.1+

![snake gif](./snake.gif)

## Install
```bash
git clone https://github.com/wick3dr0se/snake && cd ${_##*/}
```

## Usage
```bash
bash snake.sh
```

## Controls
arrow keys, or:

H - move left  
J - move down  
K - move up  
L - move right

R - restart  
Q - quit

any other key pauses
